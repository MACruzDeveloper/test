## Env Setup
  - node: `22.12.0`
  - npm: `10.9.0`

## Commands
  - `npm install` para instalar todos los paquetes
  - `npm run dev` - Arranca el proyecto en modo desarrollo en: `localhost:5173` 
  - `npm run build` - Compila todo el código Frontend en la carpeta `/dist/`

  - Sass
